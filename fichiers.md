# Fichiers


## new-item

```
new-item $work_space_file_name
```

## Set-Content

```
Set-Content $work_space_file_name '{"folders": [{"path": "."}],"settings": {}}'
```

## copy-Item 

```bash
 copy-Item -Path "C:\file.txt" -Destination "D:\newfile.txt"
 copy-Item -Path "C:\folder" -Destination "D:\" -Recurse
```
