## Split-Path

```bash

Split-Path -Path "C:\Software\Microsoft" -Parent | Split-Path -NoQualifier

```
## Test-Path

```bash
$folderPath = "C:\Example\Folder"
if (-note(Test-Path $folderPath)) {
    Write-Host "Folder does not exist"
} else {
    Write-Host "Folder exists"
    
}
```
